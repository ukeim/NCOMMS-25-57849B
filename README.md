# NCOMMS-25-57849B
Prognosis of false-negative diagnosis of melanoma patients
False negative diagnosis of melanoma – the NCOMMS-25-57849B study
Overview
This repository contains all data, codes, and outputs associated with the statistical analyses and figure generation for the study “False negative diagnosis of melanoma – the NCOMMS-25-57849B study”.
The materials provided here enable full transparency and reproducibility of the analyses and figures presented in the manuscript.
________________________________________
Data
The Data directory contains all datasets used for statistical analyses as well as for the generation of Figure 1.
•	All datasets are de-identified prior to analysis. 
•	Processed datasets used for statistical modelling are included to ensure reproducibility. 
•	Raw data structure and variable definitions are also documented.
________________________________________
Figures
The Figures directory contains all materials required to reproduce Figure 1 of the manuscript.
This includes:
•	The final publication-ready image file 
•	Corresponding source data used for figure construction 
•	The full script used to generate Figure 1 from the underlying dataset 
These materials ensure complete reproducibility of the figure generation workflow.
________________________________________
Results
The Results directory contains all statistical output tables reported in the manuscript.
This includes:
•	Descriptive statistical summaries 
•	Model outputs 
All results are directly traceable to the provided analysis scripts.
________________________________________
Codes
The Code directory contains all analysis code used to perform statistical analyses and generate figures.
This includes scripts for:
•	Descriptive statistical analyses 
•	Inferential statistical testing 
•	Figure generation 
All scripts are intended to be executed in sequential order to fully reproduce the reported results.
________________________________________
Reproducibility Statement
All analyses can be fully reproduced using the provided datasets and scripts.
Users should ensure that file paths are correctly adjusted prior to execution.
________________________________________
Notes
This repository is linked to the manuscript “False negative diagnosis of melanoma – the NCOMMS-25-57849B study” submitted to Nature Communications.

Data Availability Statement
The datasets generated and/or analysed during the current study are available within this repository. All data used for statistical analyses and for the generation of Figure 1 are provided in the Data directory.
Due to ethical and data protection regulations, individual-level clinical data cannot be publicly shared in identifiable form. All datasets included in this repository have been fully de-identified prior to analysis in accordance with institutional and ethical guidelines.
Aggregated data and processed datasets required to reproduce the reported statistical analyses are available to ensure full reproducibility of the results.
________________________________________
Code Availability Statement
All statistical analyses and figure generation procedures are fully reproducible using the codes provided in the Code directory of this repository.
The analysis was conducted using IBM SPSS Statistics (Version 28). Corresponding SPSS syntax files are included and can be executed sequentially to reproduce all reported results, including descriptive statistics, inferential analyses, and Figure 1 generation.
All scripts necessary to reproduce the analyses are publicly available within this repository.
